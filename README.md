## Build Instructions

### Prerequisites

- CMake 3.11 or higher
- A C++17-compatible compiler (e.g. clang++, g++, MSVC(for windows))
- Git

### Build Steps

- Clone the repo with:
```
git clone https://github.com/RubenAlvarezGJ/RTOS-Sim.git
```
- Nagivate to the project directory with:
```
cd RTOS-Sim
```
- Then run the following script:

### Linux/macOS

```bash
./build.sh
```

### Windows
```bash
./build.ps1
```
